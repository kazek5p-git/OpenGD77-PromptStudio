FFmpeg notice

This project may bundle a Windows ffmpeg.exe binary in release builds so users do not need to configure an ffmpeg path manually.

The bundled binary used for the 0.3.5 build reports:
ffmpeg version 6.0-essentials_build-www.gyan.dev
configuration includes --enable-gpl and --enable-version3.

FFmpeg project: https://ffmpeg.org/
Windows build source: https://www.gyan.dev/ffmpeg/builds/
